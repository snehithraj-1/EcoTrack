# EcoTrack Frontend

React + Vite frontend for the EcoTrack carbon footprint dashboard.

## Run

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 5173
```

## Scripts

```bash
npm run lint
npm run build
npm run preview
```

## Environment

Copy `.env.example` to `.env` when using real Firebase or a custom backend URL. Without Firebase variables, the app runs in demo mode.
